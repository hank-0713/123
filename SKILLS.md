# ERP 查詢技能說明

本系統共有 6 個查詢技能（tool），每個技能對應 SQL Server 中的一個 View。
主管用自然語言提問，Claude 自動選擇合適的技能並回傳結果。

---

## 1. 採購明細表 `query_purchase_orders`

**查詢內容**：採購訂單狀態、預定交期、採購量、已進貨量、未交量

**適合問題範例**：
- 下周有哪些採購單預定到期？
- 目前還有哪些訂單未完全交貨？
- USB 連接器的採購狀況如何？

**可篩選條件**：
| 參數 | 說明 |
|------|------|
| `date_from` / `date_to` | 採購日期區間（YYYYMMDD） |
| `delivery_date_from` / `delivery_date_to` | 預定交期區間（YYYYMMDD） |
| `product_name` | 產品名稱關鍵字 |
| `product_code` | 產品編號 |
| `pending_only` | true = 只顯示未交量 > 0 |

> 注意：此表無廠商資訊，廠商相關查詢請用「進貨明細表」。

## 2. 進貨明細表 `query_goods_receipts`

**查詢內容**：實際進貨記錄，含廠商名稱、產品、數量、單價、金額

**適合問題範例**：
- 本月各廠商進貨了多少金額？
- 超眾科技最近進了哪些貨？
- 本年度進貨總金額？

**可篩選條件**：
| 參數 | 說明 |
|------|------|
| `date_from` / `date_to` | 進貨日期區間（YYYYMMDD） |
| `vendor_name` | 廠商名稱關鍵字 |
| `vendor_code` | 廠商編號 |
| `product_name` | 產品名稱關鍵字 |
| `product_code` | 產品編號 |

---

## 3. 應收明細表 `query_accounts_receivable`

**查詢內容**：客戶應收帳款、已收款金額、應收餘額

**適合問題範例**：
- 目前應收帳款餘額合計多少？
- 哪些客戶還有未收的款項？
- 三月份的應收帳款情況？

**可篩選條件**：
| 參數 | 說明 |
|------|------|
| `customer_name` | 客戶名稱關鍵字 |
| `customer_code` | 客戶代碼 |
| `month_from` / `month_to` | 帳款月份區間（YYYYMM） |
| `outstanding_only` | true = 只顯示應收餘額 > 0 |

---

## 4. 科目餘額表 `query_account_balances`

**查詢內容**：各會計科目的當前餘額（資產、負債、現金、銀行）

**適合問題範例**：
- 銀行帳戶目前有多少錢？
- 公司目前的現金部位？
- 110 開頭的資產科目餘額？

**可篩選條件**：
| 參數 | 說明 |
|------|------|
| `account_code` | 科目代碼前綴（例如 `11` = 現金銀行類） |
| `account_name` | 科目名稱關鍵字（例如 `銀行`） |

**常用科目代碼**：
- `110006`：銀行存款
- `1100xx`：現金類
- `1101xx`：銀行存款類

---

## 5. 損益表 `query_income_statement`

**查詢內容**：收入與費用傳票明細，可按月份、部門、利潤中心分析

**適合問題範例**：
- 本年度銷售收入是多少？
- 四月份的費用有哪些？
- 各部門的損益狀況？

**可篩選條件**：
| 參數 | 說明 |
|------|------|
| `year` | 查詢整年（例如 `2026`） |
| `month_from` / `month_to` | 傳票月份區間（YYYYMM） |
| `account_category` | 科目大類：`4`=收入，`5`=費用 |
| `department` | 部門代碼 |
| `profit_center` | 利潤中心 |
| `account_name` | 科目名稱關鍵字 |

---

## 6. 未核准傳票 `query_unapproved_vouchers`

**查詢內容**：目前待審核的會計傳票總筆數

**適合問題範例**：
- 目前有幾張傳票還沒核准？
- 需要審核的傳票數量？

**無需輸入參數**，直接回傳數量。

## 日期說明

Claude 會自動將口語日期轉換為資料庫格式：

| 口語 | 轉換結果 |
|------|----------|
| 下周 | 下週一～下週日（YYYYMMDD） |
| 本月 | 當月 YYYYMM |
| 本年度 | YYYY01 ～ YYYY12 |
| 今天 | 當日 YYYYMMDD |

## 擴充方式

如需新增查詢技能：
1. 在資料庫新增 View
2. 在 `tools/` 新增對應 Python 檔案
3. 在 `tools/__init__.py` 註冊
4. 重新部署即生效
